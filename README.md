## 📱 Using the Application

Once the application starts, it runs in the background. Look for the icon in your system tray. Click the icon to open the main menu.

### SMS Messaging ✉️

The tool allows you to send and receive text messages directly through your modem. 

- To read messages, select "Inbox" from the menu.
- To write a message, select "New SMS," enter the phone number, type your text, and click "Send."

### SIM Security 🔐

You can manage the security features of your SIM card to prevent unauthorized use. 

- Use the PIN management window to enable or disable the SIM PIN.
- If you forget your PIN, the interface provides a way to enter your PUK code to unlock the card.

### Network Settings 📡

The application detects your cellular network automatically. It displays your current signal strength and the name of your service provider. 

- If you need to refresh your connection, click the "Reconnect" button.
- Experimental features like VoWiFi allow for voice calls over a wireless network. Toggle these features in the "Advanced" tab if your service provider supports them.

## 🛠️ Troubleshooting

If the application fails to connect to your modem, follow these steps:

1. Unplug the USB modem.
2. Wait ten seconds.
3. Plug the modem back into a different USB port.
4. Restart the application.
5. Check that your SIM card is seated correctly in its slot.

If issues persist, ensure the device drivers for your modem are installed properly. Windows usually installs these automatically, but you can check in the Device Manager. Look under "Ports (COM & LPT)" to see if the Quectel device appears.

## ⚙️ Advanced Features

The toolbox supports eSTK and eUICC functions for advanced users who need to manage multiple network profiles. Use the "eSIM Manager" tab to switch between profiles or to download new credentials provided by your carrier. Keep in mind that these settings change how your device interacts with the local cell tower.
